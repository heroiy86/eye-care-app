# Eye Care & Training (眼精疲労ケア・トレーニング)

デスクワークやスマートフォンの操作で酷使された目をケアするための、ブラウザで動作するSPA（Single Page Application）ツールです。

## 主な機能

### 1. ガポール・パッチ・トレーニング (Gabor Patch)
- **概要**: 脳の視覚処理能力を刺激し、視力回復を補助するとされるトレーニング。
- **使い方**: ぼやけた縞模様を眺めることで脳を活性化させます。クリックするとパッチが再配置されます。

### 2. 3Dステレオグラム (Stereogram)
- **概要**: 平行法・交差法を用いて、目のピント調節機能をトレーニングします。
- **使い方**: ガイドの赤い点を目安に、幾何学模様から立体図形を浮かび上がらせるように眺めます。

### 3. 遠近・視点移動トレーニング (Focus Shifting)
- **概要**: 画面内を移動する緑のターゲットを目で追うことで、毛様体筋をほぐします。
- **使い方**: 頭を動かさず、滑らかに動くターゲットを視線だけで追ってください。

### 4. リラックス・オーディオ (Relax Music)
- **概要**: トレーニングをより効果的に行うためのヒーリングBGM再生機能。

## 技術スタック
- **Frontend**: HTML5, Vanilla JavaScript
- **Styling**: Tailwind CSS (CDN利用)
- **Graphics**: Canvas API

## 公開方法
GitHub Pagesの設定で `main` ブランチを公開ソースに指定することで、即座にWebアプリとして公開可能です。
